## 🚀 Description

A simple and responsive countdown timer web app built using **HTML, CSS, and JavaScript**.  
It counts down the time remaining until the New Year in real-time.

---

## ✨ Features

- Live countdown (days, hours, minutes, seconds)
- Auto-updating every second
- Clean and responsive UI
- Built with vanilla JavaScript (no frameworks)
